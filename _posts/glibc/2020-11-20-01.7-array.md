# 数组

数组表示同类型的多个量的有序集合，数组的声明格式如下：

```
[const] <type> <name>\[num\];
```

比如

```c
int ar[k];
```

就声明了一个包含 k 个 int 类型整数的数组。

num 可以是变量，常量或字面值。若数组是全局变量，则 num 只能是字面值。

常量数组在声明后，全部数组元素是只读的，因此常量数组在声明时进行初始化才是有意义的。变量数组也可以在声明时初始化。数组初始化赋值的格式如下：

```
{ <ob_1>, <ob_2>, ... , <ob_x> }
```

例如：

```c
float answer[2] = { 1.7, 2.9 };
```

数组在声明之后，就不能用这种格式对整个数组进行赋值了，只能对单个元素赋值。像下面这样的写法是错误的：

```c
char my_name[16];
my_name = { 'c', 'a', 'l' };	/* wrong! */
```

对数组内单个元素的引用格式如下：

```
<name>\[index\]
```

index 是从 0 开始的，也就是说，它的取值范围是 [0, num-1]。对于前面已经初始化了的 answer 数组：

```c
answer[0];	/* 1.7 */
answer[1];	/* 2.9 */
```

要注意的是，如果序号越界了，编译不会报错，运行时可能不会报错。甚至序号可以是负数。至于 answer\[-1] 或 answer\[2] 表示的是什么，需要更深入的学习。目前阶段，请在引用数组元素时把序号限制在数组范围内。

不光是在 C 语言里，在日常生活中，我也希望读者能养成这样的习惯：计数从 1 开始，标号从 0 开始。如果为 10 个物品标记，从 0 开始的话是 0 ~ 9，只需 1 位；而从 1 开始的话就要两位。

下面我们应用数组来写一个程序。

```c
/*
	Array example:
	Fibonacci multiply
	lattice algorithm
	aurther: CH3CO3H
*/
#include <stdio.h>

int main(void) {
	unsigned char num1[3]={6, 2, 4};
	unsigned char num2[3]={7, 5, 8};
	unsigned char rlt[6]={0};

	unsigned char hold=0;
	for (int k=0;k<6;k++) {
		unsigned char r=0;
		for (int i=0;i<3;i++) {
			for (int j=0;j<3;j++) {
				if (i+j==k) {
					r+=num1[i]*num2[j];
					break;
				}
			}
		}
		r+=hold;
		rlt[k]=r%10;
		hold=r/10;
	}
	
	for (int k=0;k<6;k++) {
		printf("%hhu", rlt[5-k]);
	}
	printf("\n");
	
	return 0;
}
```

这个程序计算了 426\*857，最终输出答案 365082。
