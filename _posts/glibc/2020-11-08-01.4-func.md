# 函数

函数是 C 语言的核心概念，可以说，编写 C 语言，就是在实现一个个函数。前面所讲的表达式和流程控制必须要写在函数体内，而变量声明可以写在函数体外，也可以写在函数体内。

## 函数相关基本概念

我们先来看一下数学里的函数。数学函数的定义是由集合到集合的单射（不包括多值函数）。函数可以有一个或多个自变量，有定义域，函数关系式这些要素。来看下面这个函数：

```
dlt_E=f(v, p)
f(v, p)=v*L-p*v*(V_gm-V_lm)
v in [1, 20] and v in N
p in [1.013, 5.065]*10^5
```

第一行的含义是，函数值 `dlt_E` 是自变量 v 和 p 的函数，这个函数叫 f，但我们还不知道 f 具体是什么。第二行给出了函数 f 的具体形式。对于这个函数来说，L, V\_gm 和 V\_lm 都是常量。第三行和第四行给出了定义域。这个函数计算了相变过程的内能增加。

与之对应的，程序里的函数也包含两个相关概念：「函数声明」和「函数定义」。它们的格式如下：

```
<type> <function_name>([<args>]);
<type> <function_name>([<args>]) {
	...
}

<args>=<type1> <arg1>[, <type2> arg2>[...]]
```

和数学函数不同的是，函数声明不需要提供一个变量来做函数值，因为函数本身就返回了一个值，它不需要与具体的变量绑定，但我们必须给定返回值的类型。

然后我们要给函数起一个名字，函数名也属于「标识符」。我们还可以把函数声明理解成声明了一个函数类型的变量，函数名就是变量名。

函数名后跟一个小括号，里面可以有参数列表，也可以为空，就是说这个函数不需要传参数。参数类似于数学函数里的「自变量」。

我们用程序的函数来实现上面例子。

函数声明：

```c
float internal_energy_inc(unsigned char v, float p);
```

函数定义：

```c
float internal_energy_inc(unsigned char v, float p) {
    return v*L-p*v*(V_gm-V_lm);
}
```

需要注意的是，程序里的函数是可以没有返回值的。因为程序里的函数不仅仅是为返回一个值，它也有可能进行了其它操作，导致一些数据产生变化。当我们不需要函数返回一个值的时候，我们就在返回值类型那里写一个 void。void 是 C 语言里的一个关键字，除了对返回值类型占位，这个关键字还有其它的用途。比如当函数不需要传递参数给它时，我们可以在圆括号里留空，也可以写一个 void。

我们来把上一节里的阶乘计算写成函数。

```c
unsigned factorial(unsigned k);

unsigned factorial(unsigned k) {
    unsigned ret=1;
	for (unsigned i=k;i>1;i--) {
    	ret*=i;
	}
    return ret;
}
```

return 是 C 语言中的保留字，它表示立即返回，结束函数的执行，若后面跟着一个变量、常量或字面值，就将其作为函数的返回值。

函数调用，就是传给函数所需要的参数，真正去执行一个函数并得到结果，比如我想使用上面的阶乘函数计算 7 的阶乘，就可以这样去调用函数：

```c
factorial(7);
```

更多的时候，我们传给函数的参数是变量。

```c
unsigned ar;
// assign a value somehow
factorial(ar);
```

当我们需要保留下函数的返回值时，我们需要声明一个和函数返回值类型相同的变量，把函数的返回值赋给这个变量即可。

```c
unsigned ar;
// assign a value somehow
unsigned ans;
ans = factorial(ar);
```

函数调用很好理解，但有一点需要注意。当我们给函数传递变量时，我们只传递了变量的值，而与这个变量本身没有关系。如果函数相当于一台处理机器，变量相当于原材料，那么函数调用并不是把原材料直接投进机器，而是把原材料复制一份，然后把副本投入机器。也就是说，在函数调用时：

```c
factorial(a);
```

如果我们提前准确地知道，这个时候变量 a 的值为 3。那么把这条函数调用换成

```c
factorial(3);
```

不会有任何区别。

## main 函数

函数被调用的时候才会被执行，那么程序第一个调用的函数时哪个呢，是排在第一个的函数吗？并不是。C 语言程序的统一入口，就是 main 函数。

main 函数有两种原型。

```c
int main(void);
int main(int argc, const char* argv[]);
```

目前我们先使用第一种格式，不必在意第二种格式里参数的意义。另外，虽然 main 函数的返回值类型为 int，但我们只能返回 [0, 127] 的整数。

既然 main 函数是程序的统一入口，我们就不需要再去单独声明了。

再来写一个完整的程序。

```c
unsigned factorial(unsigned k);

unsigned global_var;

int main(void) {
    unsigned rlt;
    unsigned rlt2;
    
    global_var=4;
    
    rlt=factorial(6);
    rlt2=factorial(global_var);
    
    return 0;
}

unsigned factorial(unsigned k) {
    unsigned ret=1;
	for (unsigned i=k;i>1;i--) {
    	ret*=i;
	}
    return ret;
}
```

我们在 main 函数里两次引用了 factorial 函数，所传的参数分别为 6 和 4，并把计算结果保存到了变量中。

到这里，第 1 小节里的那段程序代码我们只有 `#include` 那一行和中间 printf 函数调用的部分没有理解了。
