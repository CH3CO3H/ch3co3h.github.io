# 变量与表达式

## 数据类型

每种编程语言都有各自的数据类型。所谓数据类型可以简单理解为程序中「量」的所属分类。比如数字就有整数，小数，复数这样的分类。C 语言里的数据类型分为基本数据类型和复合数据类型。

我们先来认识几个基本数据类型：

| class | meaning      |
| ----- | ------------ |
| int   | 整数         |
| float | 单精度浮点数 |
| char  | 字符         |

字符类型也是可以作为整数类型用的，因为每个字符都有唯一的编码，字符当作整数使用是所代表的数字就是编码的值。比如字符 0 代表数字 48。

在终端输入 `man ascii` 可以查看 ASCII 字符集的编码。

C 语言中没有预设的布尔类型，程序规定用整数类型代替布尔类型。整数值为 0 时表示 false，非 0 时表示 true。

## 变量与常量

程序中的「量」分为变量和常量，变量即是值可以变化的量，常量则是值不可变化的量。变量或常量需要先声明，它们的声明格式如下：

```
<type> <name>;
```

```
const <type> <name>;
```

格式里的 `name` 是我们自己给变量或常量起的名字，属于「标识符」。C 语言中的标识符只能由字母、数字及下划线构成，且不可以数字开头。数据类型名则属于 「关键字」或称「保留字」，「保留字」是不能作为标识符的，它们被编译器识别为有特殊功能的词，除数据类型外还有流程控制语句，函数返回，修饰变量、函数等等。

```c
int action_time2;
const float moment_of_inertia;
```

这样就声明了一个整数类型的名为 `action_time2` 的变量，声明了一个单精度浮点数类型的名为 `moment_of_inertia` 的常量。在后面的程序中，再次出现的变量名或常量名即表示对这个量的「引用」，即是取这个量的值。

## 表达式

表达式可以这样来定义：

- 变量或常量的引用是表达式；
- 字面值是表达式；
- 函数调用是表达式；
- 表达式经运算符恰当的连接起来是表达式。

字面值就是直接写出来的量，比如 `48` 就是整数 48，`'a'` 就是字符 "a"。另外在 C 语言中，每条语句都要以 ";" 结尾。

除返回值为空的函数调用外，表达式都有值。我们来看比较常见的赋值表达式：

```
<name> = <value>;
```

"=" 叫做赋值运算符，不叫「等号」。在数学里，「定义式」、「方程式」与「恒等式」都使用 "="，尽管在这些式子里「等号」的含义不尽相同。但在程序语言里，赋值和相等判断是不同的运算，不能弄混。在上一小节我们定义了两个量，下面我们来对其赋值。

```c
action_time2=42;
moment_of_inertia=16.4;
```

注意 `action_time2` 是变量，可以多次赋值；而 `moment_of_inertia` 是常量，只能被赋值一次，然后就不允许修改了。

赋值语句和变量声明可以整合到一起，叫做初始化表达式：

```c
int action_time5=78;
```

C 语言中的相等判断运算符写作 `==`。若为真则值为 1，否则为 0。比如下面这条语句：

```c
'A' == 65;
```

字母 A 编码为 65，因此这个表达式的值为 1。

```c
(per==1)==(tem==20);
```

上面这条表达式，在 per 等于 1 且 tem 等于20 时值为 1，在 per 不等于 1 且 tem 不等于 20 时也为 1；其余情况值为 0。

不要对浮点数做相等判断。

初学者很容易把相等判断写成赋值，为避免这种错误，有一种建议是：在将变量和常量或字面值比较时，将变量写在右侧。因为赋值语句是把右边的值赋给左边，而给字面值或已经确定的常量赋值会产生错误。来看下面的例子：

```c
char inl_let;
inl_let='B';
inl_let==98;
```

第三条相等判断若不慎错写成了赋值 `inl_let=98`，`inl_let` 的值就会变成 ‘b’。而反过来写的话，如果写错，就会变成 `98=inl_let`，给字面值 98 赋值编译器是会报错的，我们就会发现运算符写错了。

不过我个人并没有培养这种习惯，我认为反过来写语句看上去很别扭。

## 运算符的优先级

C 语言的运算符按运算对象的个数可分为「单目运算符」、「双目运算符」和三目运算符。= 是双目运算符，加、减、乘、除（+, -, *, /）都是双目运算符。逻辑取反（!）和按位取反运算（~)都是单目运算符。

一条 C 语句中出现多个运算符，运算的先后顺序叫做优先级。比如下面这条语句：

```c
jc + m * d * d;
```

程序是按照书写顺序先计算 jc + m，求和结果再进行后面的乘法呢，还是像数学中的运算一样，先乘除后加减呢？C 语言规定运算的优先顺序同数学一样，先乘除，后加减，所以对于这条语句，程序先计算 m\*d\*d，再把计算结果加上 jc。C 语言中还有很多其他的运算符，但是这里建议大家，不要去记它们的优先级，而只需要掌握这条原则：

单目运算符 > 双目运算符 > 三目运算符

其他一律使用圆括号，这样增加代码的可读性。

部分运算符的释义：

| operator | usage | means |
| -------- | ----- | ---- |
| \+=      | a\+=b |a=a+b, \-, \*, / 类推|
| ++ | a++ |a=a+1, \-\- 类推|
| % | a%b |a mod b|
