# 附录：数据类型

数据类型详表：

| class          | class       | systematic name       | other name          | range                  |
| -------------- | ----------- | --------------------- | ------------------- | ---------------------- |
| integers       | unsigned    | unsigned char         |                     | [0, UCHAR_MAX]         |
|                |             | unsigned short        |                     | \[0, USHRT_MAX]        |
|                |             | unsigned int          | unsigned            | \[0, UINT_MAX          |
|                |             | unsigned long         |                     | [0, ULONG_MAX]         |
|                |             | unsigned long long    |                     | [0, ULLONG_MAX]        |
|                | \[un]signed | char                  |                     | [CHAR_MIN, CHAR_MAX]   |
|                | signed      | signed char           |                     | [SCHAR_MIN, SCHAR_MAX] |
|                |             | signed short          | short               | [SHRT_MIN, SHRT_MIN]   |
|                |             | signed int            | singed or int       | [INT_MIN, INT_MAX]     |
|                |             | singed long           | long                | [LONG_MIN, LONG_MAX]   |
|                |             | singed long long      | long long           | [LLONG_MIN, LLONG_MAX] |
| floating point | real        | float                 |                     |                        |
|                |             | double                |                     |                        |
|                |             | long double           |                     |                        |
|                | complex     | float \_Complex       | float complex       |                        |
|                |             | double \_Complex      | double complex      |                        |
|                |             | long double \_Complex | long double complex |                        |

使用复数须引入头文件 `complex.h`。

整数的范围是预先定义在 `limits.h` 里的宏，可查阅 `/usr/include/limits.h` 找到具体的值。
