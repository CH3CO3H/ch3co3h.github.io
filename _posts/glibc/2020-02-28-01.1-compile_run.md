# 编译和运行

## 基本操作

打开任意文本编辑器，输入以下内容：

```c
/* inertia.c
	example for compiling and running. */

#include <stdio.h>

double cylinder(double m, double r);

int main(void) {
	double mass=5.0;
	double radius=1.2;
	double inertia=cylinder(mass, radius);
	printf("%lf\n", inertia);
	return 0;
}

double cylinder(double m, double r) {
	return m*r*r;
}
```

保存为 inertia.c。

使用以下命令编源译文件为可执行文件，并运行。

```shell
gcc inertia.c -o inertia && ./inertia
```

可以看到终端输出的结果。

```
7.200000
```

代码中的 /\* 到 \*/ 之间的内容叫做注释，我们可以在代码的任何地方添加注释，做适当的解释说明，编译器会把注释忽略掉。除了这种多行注释格式外，还有一种单行注释。

```c
// one line exn
```

从 // 到行尾都会被编译器忽略。

源代码文件里的空行除增加可读性外无意义，可随意添加空行，也可以删除掉。
