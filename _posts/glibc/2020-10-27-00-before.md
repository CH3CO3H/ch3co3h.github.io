# 正式学习之前

## 学习编程的目的

首先，学习编程并不等于学习黑客技术。不要指望学会了编程，就可以像影视作品中的黑客那样坐在电脑前敲着键盘，窥探他人的隐私。编程实际上就是在告诉电脑执行什么指令，如何处理数据，如何计算。而电脑所擅长的事，也只是快速进行大量计算而已。或许你是因为 AI 概念的火热想学一点编程，或许你学习编程是为了方便工作或做研究，但我个人认为，学习编程最大的意义，还是将其作为学习算法时的实践手段。而算法，才是让人拓宽视野，锻炼思维，掌握新的解决问题方法的课目。

## 思想与方法

有的编程教材上会说，学习编程重在实践，鼓励大家多写，根据结果来判断自己写的是否正确。对这种想法我是不认同的。至少对于 C 语言来说，运行结果正确不见得意味着程序写的对。要写出正确的代码，还是应该多看文档，清楚地掌握每个函数的用法，可能出现的错误，需要注意的问题。比如打开文件要关闭，用完申请的内存要释放……喜欢不求甚解，用实例代码反过来了解函数的用法，并不是一个好的做法。

## 预备知识：二进制

我们先来看一下十进制。点分十进制数里，我们有"0"~"9"这十个符号来表示数量零到九，表示数量十的时候，我们不再新造符号，而是写成"10"，十位上的"1"表示"1"个十，个位上的"0"用来占位，所以符号 0 除了表示数量零，还要用来占位。依此类推，小数点左边第 n 位的 k，表示 k\*10^n-1^，右边的第 n 位上的 k 表示 k\*10^-n^。10 的整数次幂叫做权值。那么对于二进制，只是把 10 换成 2 而已。每一位上表示数量的符号就只有 0 和 1。

比如二进制的 101 表示 2^2^+2^0^=5。

在计算机内部，指令和数据都是用二进制来表示的。

但对于人来说，二进制是很不方便的。所以我们往往用十六进制来代替二进制。以小数点为界，每四位二进制可直接换成一位十六进制数，转换后两者表示的数量是相等的。比如二进制 10011100，1100=C，1001=9，那么这个二进制数写成十六进制就是 9C。

| Bin. | Hex. |
| ---- | ---- |
| 1000 | 8    |
| 1001 | 9    |
| 1010 | A    |
| 1011 | B    |
| 1100 | C    |
| 1101 | D    |
| 1110 | E    |
| 1111 | F    |

## 编程环境

如果你使用的是 Windows10 操作系统，可以在应用商店里下载某个 Linux 子系统，比如 Ubuntu。或者也可以在电脑上装双系统。

当然，在纯 Windows 下也能学习 C 语言编程。但请自行解决编译器的选择和使用问题，另外还需要找到一种查阅标准库函数手册的方法。

为什么非要在 Linux 环境下学习 C 语言？嗯……首要原因是酷——没有哪个黑客的形象是对着电脑晃鼠标。当然深度的计算机使用者单纯用键盘操作是为了效率。其次，Linux 系统就是用 C 语言写的，你可以在你的程序中直接使用系统提供的接口，实现更多的功能。随着学习的深入，你会看到操作系统相关的一些概念与 C 语言函数的结合。
